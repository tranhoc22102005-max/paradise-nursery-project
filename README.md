# Paradise Nursery Shopping Application

Đây là dự án cuối khóa xây dựng giao diện ứng dụng mua sắm cây cảnh trực tuyến (Paradise Nursery).

## Tính năng chính (Features)
- **Landing Page:** Trang chủ giới thiệu công ty và hình nền trực quan.
- **Product Listing:** Danh sách 6 loại cây cảnh được chia thành 3 danh mục.
- **Shopping Cart:** Giỏ hàng quản lý trạng thái bằng Redux, cho phép thêm, xóa, sửa số lượng và tự động tính tổng tiền.
- **Công nghệ sử dụng:** React.js, Redux Toolkit, CSS.
